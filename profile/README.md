# NNT Lab

We build the control layer for AI agents.

**Charzel** is a model-agnostic gateway that sits between AI agents and the tools they call. It enforces policy, brokers credentials, holds risky actions for human approval, and records a trace of every decision — so a tool call only reaches a real system after something says it's allowed to.

## Why

Agents are getting connected to real tools, real data, and real money faster than most teams have a way to control what those agents are allowed to do. Charzel is that control boundary: a single enforcement point between an agent deciding to act and the action actually happening.

## Status

Pre-release. The open-source gateway core (v0.1) is in active development — not public yet.
